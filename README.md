# 誰是臥底 - 結業回憶特別版

一款適合多人同樂的派對遊戲，使用手機掃描 QR Code 即可參與！

## 線上遊玩

直接開啟：**https://你的帳號.github.io/WhoIsUndercover/**

## 遊戲規則

### 角色介紹
- **平民**：拿到多數人相同的詞語
- **臥底**：拿到與平民相似但不同的詞語
- **白板**：沒有詞語，需要從描述中猜測

### 遊戲流程
1. 主持人在大螢幕開啟遊戲，設定人數
2. 玩家用手機掃描 QR Code 加入遊戲
3. 遊戲開始後，每位玩家在手機上查看自己的詞語
4. 輪流用一句話描述自己的詞語（不能直接說出來！）
5. 每輪結束後，投票選出最可疑的人淘汰
6. 直到找出所有臥底，或臥底存活到最後

### 勝利條件
- **平民勝利**：成功找出所有臥底
- **臥底勝利**：存活到最後，或人數與平民相等

## 本地執行

如需離線使用，可下載 `WhoIsUndercover.exe` 雙擊執行。

## 技術說明

- 純前端實作，使用 localStorage 進行狀態同步
- 支援 n8n webhook 記錄玩家資料
- QR Code 使用 api.qrserver.com 產生
